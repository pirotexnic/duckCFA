# **Использование**

### **Способ 1**

Вам придется каждый раз вводить команды 
```
cd duckCFA
python3 search7.py
```
### **Способ 2**

Вы должны ввести эти команды 
```
cd duckCFA
chmod +x duckCFA
mv duckCFA ~/bin/
```
А потом запускаете скрипт с помощью 
```
duckCFA
```
